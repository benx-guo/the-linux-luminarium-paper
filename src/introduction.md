# 简介

万里之行始于足下，计算机科学教育的学习之旅往往需要从计算机环境的基础开始。不同教育机构对环境的选择各不相同：有些标准化使用 Windows，有些使用 Linux，还有一些采用新兴的 Web 技术，从而使学生无需精通操作系统。

我们大学计算机科学课程的大部分内容都期望学生具备 Linux 知识，但在这项工作之前，我们的课程体系中实际上并没有任何系统性的 Linux 教育。缺乏结构化的 Linux 课程贯穿并制约了学生整个计算机科学学习过程 [[12](references.md#ref12)]，同时也显著增加了教学人员的负担--一些本应在早期解决的基础性理解问题，直到课程后期才被发现。

这篇论文介绍了我们为了解决这个问题而开发的一种新颖的平台：Linux Luminarium。受到 2024 年网络安全教育领域新范式兴起的启发 [[17](references.md#ref17)]，我们设计并实现了 Linux Luminarium，这是一个无需额外配置、结构清晰、以挑战为驱动的 Linux 学习平台。Linux Luminarium 包含 108 个挑战，分布在 15 个主体子模块中，内容从命令行概念的学习开始，逐步延伸到对 Linux 安全模型的高级探索。

为了最大化学习效果，Linux Luminarium 引入了多项其他 Linux 教育平台未具备的创新特新，包括基于 shell 的深度插桩机制，在学习过程中提供即时引导与反馈；在学习者产生挫败之前实时识别并纠正错误的支持机制；以及通过挑战随机化来促进对 Linux 概念的持续探索。

我们于 2024 年 5 月 1 日 向公众开放了 Linux Luminarium。截至 2025 年 7 月 1 日，该平台已为 15,394 名学习者（包括本校学生及校外用户）提供学习支持，并累计完成了 697,553 次挑战。我们还将 Linux Luminarium 作为一个教学模块，部署到本校一门计算机科学专业大三必修课程中。根据教学过程中的观察反馈，涉及 Linux 的后续学习问题显著减少。

为评估 Linux Luminarium 在教育中的有效性，我们开展了两项问卷调查（均经机构审查委员会 IRB 审核并认定为 Exempt）。其中一项调查收集了 47 名校外学习者的反馈，另一项则面向 405 名本校计算机科学专业大三学生。两项调查结果一致表明，Linux Luminarium 是一种有效的教学工具，不仅解决了学习者在使用 Linux Luminarium 之前学习 Linux 时普遍感到挫败的问题，也显著提升了其 Linux 相关知识水平。此外，调查结果还显示，我们所提出的创新设计要素在实现这些效果中发挥了关键作用。

我们的目标是使 Linux Luminarium 成为全球对 Linux 感兴趣的学习者开始其计算学习之路时的重要入门平台。为实现这一目标，Linux Luminarium 向所有用户免费开放，并将平台的全部组件以开源形式发布，使社区能够在此基础上持续推进研究与开发。
